# Knight's Adventure (Legacy Project Restoration)

## Deskripsi Project
Project ini adalah hasil *backup* dan restorasi dari game lama yang dikembangkan pada tahun 2022. Saat ini, project sedang dalam proses dipindahkan ke **Unity 6** untuk dipelajari kembali dan diperbaiki sistem kodenya.

## Status Project: [BACKUP ONLY]
Harap dicatat bahwa repo ini merupakan versi *backup* mentah. Karena beberapa kendala saat pemindahan data lama, terdapat banyak file dan aset yang hilang atau *corrupt*. Project ini **belum sepenuhnya berfungsi** secara normal.

### Isu yang Diketahui (Known Issues):
* **Bug Proyektil:** Proyektil (Fireball) musuh saat ini tidak hancur/hilang meskipun sudah mengenai player.
* **Masalah Audio:** Beberapa aset suara/SFX hilang dari *database* sehingga game berjalan tanpa suara.
* **AI Boss Pasif:** Logika pengejaran (chase logic) pada Boss belum terintegrasi dengan benar, sehingga Boss hanya diam di tempat.
* **Aset Corrupt:** Beberapa aset visual mungkin tidak ter-render dengan sempurna karena perbedaan versi engine.
